# 🆓 免费部署指南

失物招领平台（ASP.NET Core + SQLite）可部署到以下免费服务器，部署后手机随时随地访问。

---

## 方案1：Azure App Service（推荐）

微软官方，ASP.NET Core 原生支持，配置最简单。

### 部署步骤

1. 注册 [azure.microsoft.com/free](https://azure.microsoft.com/free)
2. 安装 Azure CLI：`winget install Microsoft.AzureCLI`
3. 在项目目录打开终端，运行：

```bash
# 登录
az login

# 创建免费 App Service Plan
az appservice plan create -g lostandfound -n lf-plan --sku F1 --is-linux

# 创建 Web App
az webapp create -g lostandfound -p lf-plan -n lostandfound-app --runtime "DOTNET:10.0"

# 发布
dotnet publish -c Release -o pub
cd pub && zip -r ../app.zip .
az webapp deploy -g lostandfound -n lostandfound-app --src-path ../app.zip
```

部署后访问：`https://lostandfound-app.azurewebsites.net`

> ⚠️ SQLite 需要持久化存储。在 Azure Portal → App → Configuration → Path mappings 中添加 `/home` 挂载点。

## 方案2：Fly.io（Docker 部署）

简单，一条命令部署，有香港节点。

### 部署步骤

1. 注册 [fly.io](https://fly.io)
2. 安装 flyctl：`winget install Flyctl`
3. 在项目目录运行：

```bash
flyctl launch   # 按提示操作，会检测 Dockerfile
flyctl volumes create lf_data --size 1   # 创建持久化存储
flyctl deploy
flyctl open
```

### 需要修改：在项目根目录创建 `fly.toml`

```toml
app = "lostandfound"
primary_region = "hkg"  # 香港，国内访问快

[build]
  dockerfile = "Dockerfile"

[mounts]
  source = "lf_data"
  destination = "/app"

[[services]]
  internal_port = 5000
```

## 方案3：Oracle Cloud（最强免费）

4 核 ARM CPU + 24GB 内存，完全免费永久使用。相当于一台真正的服务器，想怎么部署都行。

### 部署步骤

1. 注册 [oracle.com/cloud/free](https://www.oracle.com/cloud/free/)
2. 创建 Always Free 实例（选 ARM Ampere，4核24GB）
3. SSH 登录后：

```bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 把项目上传到服务器
# （在本地电脑执行）
scp -r . ubuntu@服务器IP:/opt/lostandfound/

# SSH 到服务器启动
ssh ubuntu@服务器IP
cd /opt/lostandfound
docker compose up -d --build
```

部署后访问：`http://服务器IP:5000`

---

## 关于 SQLite 持久化注意事项

本项目使用 SQLite 文件数据库（`LostAndFound.db`），在 Docker 部署时必须挂载持久化卷，否则容器重启数据丢失。我们的 `docker-compose.yml` 已经配置了 `lf_data` 卷。
