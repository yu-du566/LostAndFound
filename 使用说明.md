## 1. 你需要什么

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11、macOS、Linux 均可 |
| .NET SDK | 10.0（免费，微软官网下载） |
| 硬盘空间 | 至少 200MB |
| 可选 | DeepSeek API Key、百度AI Key（不填也能用，只是AI功能跳过） |

---

## 2. 安装 .NET SDK

### Windows
1. 打开 https://dotnet.microsoft.com/download
2. 下载 **.NET 10.0 SDK**（不是 Runtime）
3. 双击安装，一路"下一步"
4. 安装完打开**命令提示符**，输入以下命令验证：
   ```bash
   dotnet --version
   ```
   看到 `10.0.x` 即成功

### macOS
```bash
brew install dotnet-sdk
```
或从官网下载 .pkg 安装包

### Linux (Ubuntu)
```bash
sudo apt update
sudo apt install dotnet-sdk-10.0
```

---

## 3. 解压并打开项目

1. 将收到的 `LostAndFound.zip` 解压到任意目录
2. 记住解压路径，比如 `D:\LostAndFound`
3. 打开**命令提示符**（Win+R → 输入 `cmd` → 回车）
4. 进入项目目录：
   ```bash
   cd D:\LostAndFound\LostAndFound(3)\LostAndFound
   ```
5. 确认文件完整：
   ```bash
   dir
   ```
   应该能看到 `Program.cs`、`LostAndFound.csproj`、`Controllers`、`Views`、`Services` 等

## 4. 配置 API 密钥

编辑 `appsettings.json` 文件（用记事本打开即可）：

```json
{
  "OpenAI": {
    "ApiKey": "在这里填你的DeepSeek密钥",
    "Model": "deepseek-chat",
    "Endpoint": "https://api.deepseek.com/v1/chat/completions"
  },
  "BaiduAI": {
    "ApiKey": "在这里填你的百度API Key",
    "SecretKey": "在这里填你的百度Secret Key"
  },
  "Smtp": {
    "Server": "smtp.qq.com",
    "Port": 587,
    "Username": "你的QQ邮箱@qq.com",
    "Password": "QQ邮箱授权码",
    "FromEmail": "你的QQ邮箱@qq.com",
    "FromName": "校园失物招领系统"
  }
}
```

> ⚠️ **不填也没关系**：DeepSeek/百度留空 = AI功能自动跳过；SMTP留空 = 不发邮件。系统仍可正常使用！

---

### 如何获取密钥

| 服务 | 获取方式 | 费用 |
|------|---------|------|
| DeepSeek | https://platform.deepseek.com → 注册 → API Keys | 按量付费（很便宜） |
| 百度AI | https://console.bce.baidu.com → 图像识别 → 创建应用 | 免费 50000次/天 |
| QQ邮箱SMTP | QQ邮箱 → 设置 → 账户 → POP3/SMTP → 开启 → 获取授权码 | 免费 |

## 5. 启动应用

在项目目录下执行：

```bash
dotnet run
```

看到以下输出表示成功：

```
Now listening on: http://localhost:5171
Application started. Press Ctrl+C to shut down.
```

打开浏览器访问 **http://localhost:5171**

> 首次启动会自动创建数据库，并插入 50 条失物 + 40 条招领作为示例数据。

## 6. 功能测试

### 默认管理员账号

| 用户名 | 密码 |
|--------|------|
| `admin` | `admin123` |

### 建议测试流程

#### 第1步：注册两个测试用户
```
用户A（失主）：打开注册页 → 填写信息 → 注册
用户B（捡到者）：同上，换个用户名
```

#### 第2步：用户A 发布失物
```
登录用户A → 点击"发布失物" → 填写信息 → 上传一张图片 → 提交
→ 系统会自动用百度AI识别图片分类
→ DeepSeek AI 会自动匹配已有招领
→ 如果匹配到，跳转匹配结果页
```

#### 第3步：用户B 点击"我捡到了"
```
登录用户B → 点击"失物查询" → 找到用户A的失物 → 点击"我捡到了"
→ 自动跳转到发布招领页，信息已预填好
→ 补充捡到时间地点 → 提交
→ 如果AI匹配度 ≥85%，会弹出"帮助找回"确认页
```

#### 第4步：用户A 查看通知
```
登录用户A → 首页顶部出现通知："有人可能捡到了你的XX！"
→ 点击查看详情
```

#### 第5步：管理员审核
```
登录 admin → 后台管理 → 审核认领申请 → 通过/拒绝
→ 用户会收到 SignalR 实时通知 + 邮件
```

## 7. 手机访问

### 方式一：同一WiFi

手机和电脑连同一个WiFi，浏览器访问：

```
http://192.168.x.x:5171
```

首页底部有二维码，手机扫一下就行。

> 你的局域网IP可以在首页二维码下方看到。

### 方式二：远程访问（不同网络）

使用 Cloudflare Tunnel（免费，不需要信用卡）：

```bash
# 下载 cloudflared（只做一次）
winget install cloudflared

# 启动隧道
cloudflared tunnel --url http://localhost:5171
```

会显示一个公网 URL：
```
https://xxxx.trycloudflare.com
```

把这个 URL 填入 `appsettings.json` 的 `ExternalUrl`，重启应用，首页就会出现"远程访问二维码"。手机在任何网络（4G/5G/其他WiFi）扫这个码都能访问。

## 9. 常见问题

### Q: 启动报错 "dotnet: command not found"
**A:** .NET SDK 没安装或没加到 PATH。重新安装并重启终端。

### Q: 端口 5171 被占用
**A:** 换个端口启动：
```bash
dotnet run --urls "http://0.0.0.0:5000"
```

### Q: 手机扫二维码打不开
**A:** 检查：
1. 手机和电脑是否在同一WiFi
2. Windows 防火墙是否拦截（控制面板 → 防火墙 → 允许应用通过 → 添加 5171 端口）
3. 尝试用 `http://0.0.0.0:5171` 启动

### Q: AI 匹配/识别不工作
**A:** 检查 `appsettings.json` 中 API Key 是否正确填写。不填则自动跳过，不影响基本功能。

### Q: 如何重置数据库
**A:** 删除项目目录下的 `LostAndFound.db` 文件，重启应用即可自动重建。

### Q: 如何修改管理员密码
**A:** 用 admin 登录 → 个人信息 → 修改密码。或者删除 `.db` 文件重置，默认密码 `admin123`。
